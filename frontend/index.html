Model Selection and Performance

This application predicts insurance fraud probability using machine learning. The dataset contains 12,002 records and 28+ features, with the target variable being fraud reported. The system was trained and evaluated using multiple classification models:

  • Logistic Regression (Accuracy: 77.38%)
  • Decision Tree (Accuracy: 67.68%)
  • Random Forest (Accuracy: 77.84%)

Final Implementation

Random Forest achieved the highest performance and was selected as the final deployed model. It features a cross-validation mean score of 77.82%.

Hyperparameter tuning was performed using GridSearchCV to optimize model performance, ensuring professional, robust evaluation metrics on complex datasets.
